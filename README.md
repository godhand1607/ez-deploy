# ez-deploy

## Quick Start
1. Clone this repo
    ``` sh
    git clone https://github.com/godhand1607/ez-deploy.git
    ```
2. Go into the project's root directory
3. Run `npm install`
4. Run `npm start`
5. On your browser, access http://localhost:8080/

## Deployment platforms
1. Easiest -- surge.sh
2. Easier -- gh-pages
3. Easy -- AWS S3
